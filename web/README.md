# dev-ai-search-web

## Getting Started

### Prerequisites
- **Node.js**: v22.10.0 or higher
- **npm**

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/campus-aisearch/dev-ai-search-web.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd dev-ai-search-web
   ```

3. **Install dependencies**
   This step downloads all the necessary libraries (React, TypeScript, bcryptjs, etc.) listed in `package.json`.
   ```bash
   npm install
   ```

### Running the App

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or the port shown in your terminal).
